What is QEngine?

QEngine is a game engine written entirely in C#. It uses Avalonia for rendering, allowing applications to run across multiple platforms.

The engine was designed with a focus on simple and direct game development, without relying on a built-in graphical editor. QEngine does not use a visual GUI editor, unlike popular engines such as Unity, Unreal Engine, or Godot.

Project management is handled through the command line, using dedicated CLI commands responsible for:

creating projects,

searching for projects on disk,

building applications,

running games.

QEngine is actively developed, with its features continuously improved and expanded.

The API may change in future versions of the engine.
