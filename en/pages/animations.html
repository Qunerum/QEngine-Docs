Types and Classes

If the Animator is not assigned to a GameObject, it will not work.

Animation

Public Functions:

Animation.AddFrame(Sprite frame) // Adds a frame to the animation.
Animation.AddFrames(List<Sprite> frames) // Adds frames to the animation.
Animation.RemoveFrame(int index) // Removes the frame at the given index.
            
Animation.GetFrame(int index) // Returns the frame at the given index.
Animation.GetFrames() // Returns all animation frames as a List<Sprite>.
Animation.GetFramesCount() // Returns the total number of animation frames.
            
Animation.GetFPS() // Returns the number of frames (animation) played per second.
Animation.SetFPS(int fps) // Sets the number of frames (animation) played per second.
            
Animation.Clear() // Removes all frames from the animation.

Usage:

Animator anim = new Animator();
// Creates a new Animator not assigned to any object.

How to assign to a GameObject?

GameObject obj = new GameObject();
Animator anim = obj.AddComponent<Animator>();

Animator

Public Fields:

Animator.isPlaying // bool
Animator.isPaused // bool

Public Functions:

Animator.GetAnimation(string name) // Returns the animation with the given name.
Animator.AddAnimation(string name) // Adds an empty animation.
Animator.RemoveAnimation(string name) // Removes the animation with the given name.
            
Animator.AddFrame(string name, Sprite frame) // Adds a frame to the specified animation.
Animator.AddFrames(string name, List<Sprite> frames) // Adds frames to the specified animation.
            
Animator.Play(string name) // Starts playing the selected animation.
Animator.Pause() // Pauses animation playback.
Animator.UnPause() // Resumes animation playback.
Animator.Reset() // Resets the animation to the first frame.

Usage:

Animator anim = new Animator();
// Creates a new Animator not assigned to any object.

How to assign to a GameObject?

GameObject obj = new GameObject();
Animator anim = obj.AddComponent<Animator>();
