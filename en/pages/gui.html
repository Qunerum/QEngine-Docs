Types and Classes

If a UI element is not assigned to a GameObject, it will not be displayed.

UI elements respond to input events from the QEngine.Input system.

Color

Public Fields:

Color.r // byte
Color.g // byte
Color.b // byte
Color.a // byte

Constructors:

new Color() // Creates a new color (white).
new Color(byte rgb) // Creates a new color, sets 'r', 'g', and 'b' to 'rgb' (a = 255).
new Color(byte rgb, byte a) // Creates a new color, sets 'r', 'g', and 'b' to 'rgb'.
new Color(byte r, byte g, byte b) // Creates a new color with specified values (a = 255).
new Color(byte r, byte g, byte b, byte a) // Creates a new color with specified values.

Usage:

Color color = new Color(100); 
// Creates a new color (100, 100, 100, 255) (gray)

Image

Public Fields:

Image.size // Vector2
Image.sprite // Sprite
Image.color // Color

If the sprite is not set, a rectangle of the given size will be displayed on the scene.

Usage:

Image img = new Image();
// Creates a new Image not assigned to any object.

How to assign to a GameObject?

GameObject obj = new GameObject();
Image img = obj.AddComponent<Image>();
