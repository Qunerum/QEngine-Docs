Input

Input provides access to keyboard input and mouse position.

Public fields:

Input.mousePosition // Vector2Int
// Cursor position in the game window.

Input.mouseLeft // bool
// Left mouse button state.

Public methods:

Input.Read(Key key) // Reads keyboard key state.
Input.ReadDown(Key key) // Reads key press state.

ReadDown returns true only on the frame the key was pressed.
