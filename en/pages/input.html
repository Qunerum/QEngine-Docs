Types and Classes

Input

Input represents reading keyboard button states and mouse position.

Public Fields:

Input.mousePosition // Vector2Int
// Cursor position in the game window.        
        
Input.mouseLeft // bool
// Left mouse button state.

Public Functions:

Input.Read(Key key) // Reads the state of a keyboard key.
Input.ReadDown(Key key) // Reads the key-down state.
      
Input.AddAxisInt(string name, Key left, Key right) // Adds a new int axis.
Input.AddAxisInt(string name, List<Key> left, List<Key> right) // Adds a new int axis.
Input.GetAxisInt(string name) // Returns the value read from the int axis.
      
Input.AddAxisVector2(string name, Key up, Key bottom, Key left, Key right) // Adds a new Vector2 axis.
Input.AddAxisVector2(string name, List<Key> up, List<Key> bottom, List<Key> left, List<Key> right) // Adds a new Vector2 axis.
Input.GetAxisVector2(string name) // Returns the value read from the Vector2 axis.

ReadDown returns true only in the frame in which the key was pressed.
