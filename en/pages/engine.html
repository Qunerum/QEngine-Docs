Vector2

Vector2 represents a two-dimensional vector or point in 2D space.

Most commonly used for positions, sizes, directions, and mathematical calculations.

Public fields:

Vector2.x // float
// X component of the vector

Vector2.y // float
// Y component of the vector

Constructor:

new Vector2(float x, float y) // Creates a new vector with the given coordinates.

Math operators:

new Vector2(8f, 1f) + new Vector2(5f, 3f) // Result: new Vector2(13f, 4f)
new Vector2(23f, 4f) - new Vector2(9.6f, 7.2f) // Result: new Vector2(13.4f, -2.8f)
new Vector2(3f, 5f) * new Vector2(9f, 6.1f) // Result: new Vector2(27f, 30.5f)
new Vector2(12f, 5f) * 3f // Result: new Vector2(36f, 15f)
new Vector2(50f, 27f) / new Vector2(5f, 3f) // Result: new Vector2(10f, 9f)
new Vector2(12f, -8f) / 4f // Result: new Vector2(3f, -2f)

Usage:

Vector2 vector = new Vector2(5.2f, 9.4f);
// Creates a vector with coordinates (x: 5.2f, y: 9.4f)

QEScene

QEScene represents a scene.

Public fields:

QEScene.cameraPosition // Vector2
// Camera position

Public methods:

QEScene.AddGameObject(GameObject obj) // Adds 'obj' to the scene.
QEScene.GetGameObjects() // Returns all objects in the scene as List<GameObject>
QEScene.GetGameObject(string name) // Returns a GameObject with the given name

The Init() method is called once when the scene is loaded.

Usage:

public class SceneExample : QEScene
{
    public override void Init()
    {
        // Executed on scene start
    }
}

GameObject

GameObject represents an object in the scene.

Public fields:

GameObject.name // string
// Object name

Public methods:

GameObject.AddComponent<T>() // Adds a component if it does not already exist.
GameObject.GetComponent<T>() // Gets a component if it exists.

bool GameObject.TryGetComponent<T>(out T component)
// Attempts to retrieve a component of type T.
// Returns true if the component exists and assigns it to 'component'.
// Otherwise returns false.

Usage:

GameObject obj = new GameObject();
// Creates a new GameObject in the current scene.

GameObject obj2 = new GameObject("hello");
// Creates a new GameObject named "hello" in the current scene.
