What is QEngine?

QEngine is a lightweight 2D game engine written in C#, designed for:

- fast prototyping,

- full code control,

- component-based architecture,

- CLI-driven workflows.

It takes inspiration from Unity, Godot, and MonoGame, while keeping things simple and minimal.
